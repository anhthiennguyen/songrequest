# DJ Session - Song Request App

A modern React application for managing DJ sessions where users can request songs and vote on them.

## Features

- Create DJ sessions with unique session IDs
- Share session links with others
- Request songs to be played
- Vote on requested songs
- Real-time song ranking based on votes

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)

## Usage

1. Click "Create DJ Session" to start a new session
2. Share the session link with your audience
3. Users can request songs and vote on them
4. Songs are automatically sorted by vote count
